# 🕒 Virtual Clock

A modern and responsive **Virtual Clock** web application built using **HTML, CSS, and JavaScript**. It combines three useful utilities in a single application:

- 🕒 Live Digital Clock
- ⏱️ Stopwatch
- ⏲️ Countdown Timer

The application features a clean user interface, responsive design, and interactive controls for an intuitive user experience.

## ✨ Features

### 🕒 Live Clock
- Real-time digital clock
- Displays current day and date
- Switch between **12-hour** and **24-hour** formats

### ⏱️ Stopwatch
- Start stopwatch
- Stop/Pause stopwatch
- Resume stopwatch
- Reset stopwatch
- Millisecond precision
- Record lap times
- Scrollable lap history

### ⏲️ Countdown Timer
- Set hours, minutes, and seconds
- Input validation
- Start countdown
- Pause countdown
- Resume countdown
- Reset timer
- Alert when countdown finishes

### 🎨 User Interface
- Responsive design
- Sidebar navigation
- Material Icons
- Modern button styling
- Hover effects
- Clean layout
